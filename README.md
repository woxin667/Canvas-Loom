# Canvas Card Actions

`Canvas Card Actions` 是一个面向 Obsidian Canvas 的增强插件，提供卡片拆分、快速复制/拼合、预览工作台、徽章标记和属性管理等常用操作，帮助你更高效地整理大纲、任务流和画布笔记。

当前版本：`1.3.0`

## 主要功能

### 拆分卡片

- 单个文本卡片右键后可看到统一入口 `拆分卡片...`
- 支持按自定义分隔符拆分
- 支持按 Markdown 标题层级拆分（一级到六级）
- 标题前的导言内容会自动归入第一张拆分后的卡片
- 更深层标题会保留在所属父级卡片中，避免内容被过度切碎

### 复制卡片内容

- `复制卡片内容`：复制单张文本卡片内容
- `一键复制`：按设置中的 `一键排序方式` 复制多张卡片
- `一键拼合`：按设置中的 `一键排序方式` 直接在当前 Canvas 新建合并卡片
- `打开预览...`：进入统一工作台，在其中切换排序、预览文本并决定输出
- 工作台支持 `位置 / 徽章 / 手动` 三种排序模式
- 徽章排序输出格式为 `[徽章] 内容`

### 预览工作台

- 作为标准 Obsidian `ItemView` 打开，默认位于侧边栏
- 支持被用户原生拖成独立窗口
- 手动模式会继承当前排序结果作为拖拽起点
- 底部预览默认折叠，展开后再生成最终合并文本
- 统一输出 `复制 / 新建卡片 / 新建文稿`

### 卡片徽章

- 支持为 Canvas 卡片添加数字、文字或 emoji 徽章
- 徽章显示在卡片右上角，适合标记优先级、状态或分类
- 徽章数据会直接持久化到 `.canvas` 文件中

### 卡片属性管理

- 单卡片和多卡片场景统一通过 `管理卡片属性` 入口进入
- 单卡片可查看尺寸、位置、徽章和内容预览，并直接编辑宽高
- 多卡片可统一尺寸、批量查看统计信息
- 提供 `复制选中卡片的尺寸` 命令，适合绑定快捷键快速使用

## 典型用法

### 1. 拆分长卡片

1. 在 Canvas 中右键单个文本卡片
2. 选择 `拆分卡片...`
3. 在弹出的选择器中挑选：
   - `按分隔符拆分`
   - `按一级标题拆分`
   - `按二级标题拆分`
   - 其他当前可用的标题层级

### 2. 整理多卡片内容

1. 在 Canvas 中选中多张文本卡片
2. 打开选择菜单
3. 根据场景选择：
   - `一键复制`
   - `一键拼合`
   - `打开预览...`
4. 如果需要精调顺序或导出 Markdown 文稿，进入工作台后再操作

### 3. 给卡片加状态标记

1. 右键任意支持的 Canvas 卡片
2. 选择 `添加/编辑徽章`
3. 输入数字、文字或 emoji

## 插件设置

可在插件设置页中配置：

- `设置Canvas卡片分隔符`：控制按分隔符拆分时使用的文本
- `设置卡片排序优先级`：控制按位置排序时优先按纵向还是横向排序
- `一键排序方式`：控制一键复制、一键拼合和工作台默认排序模式
- `启用徽章功能`：控制是否显示卡片徽章

## 安装

### 从 GitHub Releases 安装

1. 打开本仓库的 [Releases](https://github.com/Worthing667/Obsidian-Canvas-Card-Actions/releases)
2. 下载发布页提供的 `obsidian-canvas-card-actions-x.y.z.zip`
3. 解压后将其中的 `main.js`、`manifest.json`、`styles.css` 放入：
   `.obsidian/plugins/obsidian-canvas-card-actions/`
4. 在 Obsidian 的社区插件设置中启用本插件

### 手动构建

```bash
npm install
npm run build
```

构建完成后可在 `build/` 目录中找到插件文件。

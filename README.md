# Canvas Card Actions

本插件为 Obsidian Canvas 提供了多项增强功能，包括拆分卡片、复制内容、添加徽章以及卡片属性管理。您可以根据自定义分隔符拆分卡片，方便地复制单个或多个卡片的内容，还可以为卡片添加直观的状态或优先级徽章，并管理卡片的尺寸和位置属性。

## 如何使用

### 拆分卡片功能

1. 在 Canvas 视图中，右键点击一个文本卡片。
2. 从上下文菜单中选择"按分隔符拆分卡片"。
3. 插件将根据您在设置中指定的分隔符来拆分卡片内容，并在原始卡片右侧创建新的卡片。

### 复制卡片内容功能

- **复制单个卡片**：右键点击一个文本卡片，选择"复制卡片内容"即可将卡片内容复制到剪贴板。
- **复制多个卡片**：选中多个卡片，在任意选中卡片上右键选择"复制已排序的卡片内容"，插件会按从上到下、从左到右的顺序复制卡片内容。
- **按徽章顺序复制**：选中多个带徽章的卡片，右键选择"按徽章顺序复制内容"，插件会按徽章内容的顺序复制卡片内容。

### 卡片徽章功能

1. 在 Canvas 视图中，右键点击任意卡片（文本、文件、链接等）。
2. 选择"添加/编辑徽章"。
3. 在弹出的窗口中输入徽章内容（如数字、文字或 Emoji），即可为卡片添加一个醒目的徽章，用于状态或优先级标记。
4. 徽章数据会自动保存在 Canvas 文件中，实现持久化。

### 卡片属性管理功能

- **单卡片属性管理**：右键点击单个文本卡片，选择"编辑卡片尺寸..."可以查看和编辑卡片的尺寸、位置等属性。
- **多卡片属性管理**：选中多个文本卡片，右键选择"批量管理卡片属性..."可以统一管理多个卡片的尺寸和位置属性。
- **批量尺寸调整**：
  - 统一为最小/最大/平均尺寸
  - 自定义尺寸（支持只调整宽度或只调整高度）
  - 锁定宽高比等比例调整
- **属性复制**：可以复制卡片的尺寸、位置或完整属性信息到剪贴板。

## 设置

- **Canvas 卡片分隔符**: 您可以在插件的设置页面中，设置用于拆分卡片的分隔符。默认值为 `---`。

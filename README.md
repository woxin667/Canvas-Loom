# Canvas Card Actions

本插件为 Obsidian Canvas 提供了多项增强功能，包括拆分卡片、批量复制内容、添加徽章以及卡片属性管理。您可以根据自定义分隔符轻松拆分长卡片，方便地按顺序或徽章优先级复制多个卡片内容，为卡片添加直观的状态徽章，并批量管理卡片的尺寸和位置属性。

## 💡 主要功能与使用方法

### ✂️ 拆分卡片功能

1. 在 Canvas 视图中，右键点击一个包含文本的卡片。
2. 从上下文菜单中选择 **"按分隔符拆分卡片"**。
3. 插件将根据您在设置中指定的分隔符来拆分卡片内容，并在原始卡片右侧自动创建新的卡片序列。

### 📋 复制卡片内容功能

- **复制单个卡片**：右键点击单个文本卡片，选择 **"复制卡片内容"**，即可将所选内容存入剪贴板。
- **按位置复制内容**：选中多个卡片，右键点击并选择 **"按位置复制内容"**，插件会根据设置中的排序优先级（例如从上到下、从左到右）合并并复制所有选中卡片的内容。
- **按徽章顺序复制**：选中多个带有徽章标记的卡片，右键选择 **"按徽章顺序复制内容"**，插件将依照徽章标识的顺序依次组织并复制内容。

### 🏷️ 卡片徽章功能

1. 右键点击 Canvas 上的任意卡片目标（文本、文件或网页链接等）。
2. 从菜单中选择 **"添加/编辑徽章"**。
3. 在弹窗内输入徽章内容（如优先级数字 `1`，状态文字 `TODO` 或表情符号 `🔥`），卡片上会立刻显示相应的醒目徽章。
4. 徽章信息直接存储在 Canvas 数据中，完全持久化。

### ⚙️ 卡片属性管理功能

- **管理卡片属性**：无论您选中单张还是多张卡片，右键菜单中都会提供统一的 **"管理卡片属性"** 选项，您也可以配置快捷键快速使用。在属性管理面板中您可以：
  - **批量调整尺寸**：统一更改宽度/高度，支持对齐多卡片中的最大、最小或平均尺寸。
  - **锁定比例**：提供锁定宽高比的等比例缩放设定。
- **复制卡片尺寸**：您可以通过命令面板或快捷键执行 **"复制选中卡片的尺寸"**，将尺寸属性复制到剪贴板。

## 🔧 插件设置

在 Obsidian 的插件设置页面，您可以对 Canvas Card Actions 进行以下定制：

- **设置Canvas卡片分隔符**: 设定拆分卡片时使用的标识符，默认为 `---`。
- **设置卡片排序优先级**: 决定批量复制多张卡片时的文本合并顺序：
  - 优先按垂直方向排序（从上到下，然后从左到右）
  - 优先按水平方向排序（从左到右，然后从上到下）
- **启用徽章功能**: 全局控制是否在 Canvas 视图中显示徽章。

# 徽章功能函数调用关系

## 类结构

### BadgeModal（模态框类）
独立的模态框类，负责显示徽章输入界面

#### 方法：
- `constructor(plugin, node, currentBadge)` - 构造函数
- `onOpen()` - 打开模态框时的初始化
- `setBadge(badgeText)` - 设置徽章的核心逻辑
- `onClose()` - 关闭模态框时的清理

### Cardify（主插件类）
主插件类，包含所有核心功能

#### 方法：
- `addBadgeCommand(menu, node)` - 添加徽章命令到右键菜单
- `getCurrentBadge(node)` - 获取卡片当前徽章内容
- `updateNodeBadgeDisplay(node)` - 更新节点徽章显示
- `initializeAllCanvasBadges()` - 初始化所有 Canvas 徽章显示
- `onload()` - 插件加载时的初始化

## 函数调用关系图

```
onload()
├── 注册右键菜单事件监听器
│   ├── canvas:node-menu 事件
│   │   └── addBadgeCommand(menu, node)
│   │       └── new BadgeModal(plugin, node, currentBadge).open()
│   │           └── getCurrentBadge(node)
│   └── canvas:selection-menu 事件
│       └── addBadgeCommand(menu, node)
│           └── new BadgeModal(plugin, node, currentBadge).open()
│               └── getCurrentBadge(node)
├── 注册文件变化监听器
│   └── metadataCache.on("changed")
│       └── updateNodeBadgeDisplay(node)
└── 注册布局就绪回调
    └── initializeAllCanvasBadges()
        └── updateNodeBadgeDisplay(node)

BadgeModal.onOpen()
└── 创建界面元素
    └── 点击"确定"按钮
        └── setBadge(badgeText)
            ├── 直接操作 node.nodeEl 属性
            └── 显示通知

getCurrentBadge(node)
└── 从 node.nodeEl.getAttribute("data-badge") 获取徽章内容

updateNodeBadgeDisplay(node)
├── 检查设置是否启用徽章功能
├── 如果未启用，移除徽章显示属性
└── 如果启用，保持现有属性（徽章信息已直接存储在属性中）
```

## 事件流

### 1. 右键菜单触发
```
用户右键点击卡片
→ 触发 canvas:node-menu 或 canvas:selection-menu 事件
→ 调用 addBadgeCommand()
→ 获取当前徽章内容 getCurrentBadge()
→ 创建并显示 BadgeModal
```

### 2. 徽章设置
```
用户在模态框中输入内容并点击确定
→ 调用 BadgeModal.setBadge()
→ 直接设置 node.nodeEl 的 data-badge 属性
→ CSS 自动应用样式显示徽章
```

### 3. 初始化和更新
```
插件加载完成
→ onload() 中调用 initializeAllCanvasBadges()
→ 遍历所有 Canvas 节点
→ 调用 updateNodeBadgeDisplay() 更新显示状态
```

## 数据流

### 徽章数据存储
```
用户输入徽章内容
→ setBadge() 方法
→ node.nodeEl.setAttribute("data-badge", badgeText)
→ node.nodeEl.setAttribute("data-badge-type", type)
→ CSS 选择器匹配并显示
```

### 徽章数据读取
```
显示模态框时需要预填充内容
→ getCurrentBadge() 方法
→ node.nodeEl.getAttribute("data-badge")
→ 返回当前徽章内容
```

# 功能：卡片内容复制与排序

## 用途

处理单张或多张文本卡片的内容输出。多卡片场景分为“直接完成”和“进入工作台”两条路径。

相关代码：

- `src/presentation/commands/CopyCommands.ts`
- `src/presentation/commands/QuickActionCommands.ts`
- `src/services/ContentService.ts`
- `src/services/MergeService.ts`
- `src/presentation/views/MergeWorkbenchView.ts`

## 单张卡片

- 右键单张文本卡片
- 使用 `复制卡片内容`
- 直接复制纯文本，不涉及排序

## 多张卡片的快速操作

### `一键复制`

- 读取当前选区
- 按设置中的 `一键排序方式` 排序
- 直接复制到剪贴板

### `一键拼合`

- 读取当前选区
- 按设置中的 `一键排序方式` 排序
- 在当前 Canvas 新建一张合并卡片

默认排序方式支持：

- `按位置顺序`
- `按徽章顺序`

按徽章顺序输出时，文本格式为：

```text
[徽章] 内容
```

## 预览工作台

### 入口

- 多选文本卡片
- 选择 `打开预览...`

### 能力

- 切换排序模式
- 查看当前顺序
- 展开预览最终文本
- 输出到剪贴板、Canvas 卡片或 Markdown 文稿

### 排序模式

- `位置`：自动排序，只读
- `徽章`：自动排序，只读
- `手动`：以当前排序结果为起点，再允许拖拽微调

### 预览行为

- 预览区默认折叠
- 展开后才生成最终文本
- 大内容场景下可减少不必要的渲染开销

## 设置项

- `设置卡片排序优先级`
  只影响位置排序优先按纵向还是横向。
- `一键排序方式`
  影响 `一键复制`、`一键拼合` 和工作台默认排序模式。

## 适合使用工作台的场景

- 需要先检查合并结果
- 需要从自动排序继续手动调整
- 需要导出 Markdown 文稿

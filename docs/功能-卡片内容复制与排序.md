# 功能：卡片内容复制与排序

本插件增强了 Obsidian Canvas 的复制功能，提供了基于卡片空间位置的智能排序复制，并优化了单张与多张卡片的操作流程。

## 核心功能

### 1. 智能排序复制（多张卡片）

当您在 Canvas 中选中**多张卡片**时，可以通过以下方式触发智能排序复制：

- **右键菜单**：在选中的任意一张卡片上右键，选择 **“复制已排序的卡片内容”**。
- **顶部选择菜单**：选中多张卡片后，在画布顶部出现的选择工具栏中点击 **“复制已排序的卡片内容”**。

**排序逻辑：**

插件会自动按照指定的顺序对选中的卡片进行排序，然后将它们的文本内容合并复制到剪贴板。每个卡片的内容之间会用换行符分隔。

默认排序方式为从上到下、从左到右。您可以在插件设置中更改此行为：
- **优先按垂直方向排序**：从上到下，然后从左到右
- **优先按水平方向排序**：从左到右，然后从上到下

这对于需要按特定顺序整理和导出思路的场景非常有用。

### 2. 单张卡片内容复制

当您只选中**一张卡片**时，菜单项会变为 **“复制卡片文本”**。

- **右键菜单**：在卡片上右键，选择 **“复制卡片文本”**。
- **顶部选择菜单**：选中卡片后，在画布顶部出现的选择工具栏中点击 **“复制卡片文本”**。

此功能会直接将该卡片的纯文本内容复制到剪贴板，方便快速引用。

## 使用场景

- **会议记录整理**：将分散记录在不同卡片上的会议要点，按视觉顺序一键复制整理成有序的文本。
- **灵感与头脑风暴**：将 Canvas 中自由排列的想法，按从上到下的逻辑顺序导出，形成结构化的草稿。
- **流程步骤说明**：将流程图中的各个步骤卡片，按顺序复制出来，生成操作指南。

# 功能：卡片徽章

## 用途

为 Canvas 卡片添加数字、文字或 emoji 标记，用于表达状态、优先级或分类。

相关代码：

- `src/presentation/commands/BadgeCommands.ts`
- `src/presentation/modals/BadgeModal.ts`
- `src/services/BadgeService.ts`
- `src/presentation/styles/BadgeStyles.ts`

## 使用方式

1. 右键文本卡片
2. 选择 `添加/编辑徽章`
3. 输入数字、文字或 emoji
4. 保存后立即显示

移除徽章时，可以清空输入内容或使用移除操作。

## 徽章类型

- 数字：适合优先级、顺序
- 文字：适合状态、标签
- emoji：适合视觉化标记

## 存储与显示

- 徽章数据直接写入 `.canvas` 节点
- 关闭或重新打开后仍会保留
- 分享 `.canvas` 文件时会一起带出
- 徽章显示在卡片右上角

## 设置项

- `启用徽章功能`
  控制是否显示徽章样式

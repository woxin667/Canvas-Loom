# 功能：为 Canvas 卡片添加/移除数字徽章

## 功能概述

此功能允许用户为 Obsidian Canvas 中的文本卡片添加数字、文字或 emoji 徽章。徽章会显示在卡片的右上角，用于快速标识卡片的状态、优先级或其他自定义信息。

## 使用方法

1. 在 Canvas 中右键点击任意文本卡片
2. 选择"添加/编辑徽章"选项
3. 在弹出的对话框中输入徽章内容（可以是数字、文字或 emoji）
4. 点击"确定"保存徽章
5. 要移除徽章，可以在对话框中清空内容或点击"移除徽章"按钮

## 徽章类型

- **数字徽章**：纯数字内容（如 "1", "3", "99"），显示为圆形蓝色徽章
- **文字徽章**：文字内容（如 "Done", "Important"），显示为圆角矩形灰色徽章
- **Emoji 徽章**：Emoji 表情（如 "✅", "🔥", "⭐"），显示为透明背景的徽章

## 存储方式

徽章信息直接存储在 Canvas 卡片元素的 `data-badge` 属性中，不依赖于关联的笔记文件：
- 避免了创建不必要的文件
- 徽章类型通过 `data-badge-type` 属性标识（number/text/emoji）

## 设置选项

在插件设置中，可以启用或禁用徽章功能：
- **启用徽章功能**：控制是否在 Canvas 卡片上显示徽章

## 技术实现

1. **数据存储**：直接在卡片元素的属性中存储徽章信息
2. **界面交互**：通过右键菜单提供添加/编辑徽章的功能
3. **视觉呈现**：使用 CSS 伪元素在卡片右上角显示徽章，通过动态样式注入确保样式正确应用
4. **实时更新**：设置徽章后立即在卡片上显示

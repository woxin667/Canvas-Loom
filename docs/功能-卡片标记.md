# 功能：卡片标记

## 用途

为 Canvas 卡片添加数字序号标记，用于排序、编号和后续内容整理。

相关代码：

- `src/presentation/commands/BadgeCommands.ts`
- `src/presentation/modals/BadgeModal.ts`
- `src/services/BadgeService.ts`
- `src/presentation/styles/BadgeStyles.ts`

## 使用方式

1. 右键文本卡片
2. 选择 `编辑标记`
3. 输入数字序号，例如 `1`、`2.1`、`10.3.2`
4. 保存后立即显示

移除标记时，可以清空输入内容或使用移除操作。

## 输入规则

- 仅支持数字序号
- 正则规则：`/^\d+(\.\d+)*$/`
- 允许层级编号，例如 `1.2`、`2.10.3`
- 非法输入会在弹窗内直接拦截，不能提交

## 存储与显示

- 标记数据直接写入 `.canvas` 节点的 `badge` 字段
- 关闭或重新打开后仍会保留
- 分享 `.canvas` 文件时会一起带出
- 显示为卡片右上角药丸标记
- 标记样式统一定义在插件的 `styles.css` 中，并使用 Obsidian 主题变量跟随当前主题

这样处理的原因是：

- 标记显示逻辑和样式来源更稳定，也更容易维护

## 设置项

- `启用标记功能`
  控制标记显示开关。

当关闭时：

- 停止给当前画布节点补显示用的 `data-badge`
- 跳过标记加载
- 清理 DOM 上已有的 `data-badge`

当打开时：

- 重新把 `.canvas` 中已有标记加载到当前画布

# 功能：拆分 Canvas 卡片

## 用途

将单张文本卡片按规则拆成多张卡片，适合整理长笔记、会议记录和分层提纲。

## 入口

- 单张文本卡片右键
- 选择 `拆分卡片...`
- 在弹出的选择器中执行具体拆分方式

相关代码：

- `src/main.ts`
- `src/presentation/modals/SplitCardModal.ts`
- `src/services/CardService.ts`

## 支持的拆分方式

### 按分隔符拆分

- 使用设置中的 `canvasCardDelimiter`
- 清理空白段落后再拆分
- 第一段保留在原卡片
- 其余段落创建为新卡片

### 按标题层级拆分

- 识别 `#` 到 `######`
- 只把选定层级的标题当作新卡片起点
- 更深层标题保留在所属父级内容中
- 标题前的导言内容会归入第一张拆分后的卡片

## 选择器行为

- `按分隔符拆分` 始终显示
- 检测到分隔符时可点击，并显示预计拆分数量
- 未检测到分隔符时显示禁用提示
- `按标题拆分` 能力始终显示
- 没有可用标题层级时显示禁用提示，不隐藏功能

## 新卡片规则

- 原卡片更新为第一段内容
- 新卡片继承原卡片宽高
- 新卡片按固定间距排在原卡片右侧
- 创建完成后调用保存逻辑写回 `.canvas`

## 限制与反馈

- 拆分结果少于 2 张卡片时不会执行
- 非文本卡片不会显示此入口
- 分隔符由插件设置控制，默认值为 `'---'`

# 功能：按徽章顺序复制内容

## 功能概述

"按徽章顺序复制内容"是Canvas Card Actions插件的核心功能之一，允许用户根据卡片的徽章标记来复制和排序卡片内容。该功能特别适用于项目管理、任务规划等需要按优先级或标识符组织信息的场景。

## 使用方法

### 基本操作流程

1. **选择卡片**：在Canvas中选中多张带有徽章的文本卡片
2. **右键菜单**：右键点击选中的卡片区域
3. **选择功能**：点击"按徽章标记顺序复制内容"菜单项
4. **获得结果**：内容已按徽章顺序复制到剪贴板

### 输出格式

复制的内容格式为：
```
[徽章1] 卡片内容1

[徽章2] 卡片内容2

[徽章3] 卡片内容3
```

### 使用示例

假设Canvas中有以下带徽章的卡片：
- 卡片A：徽章"3"，内容"完成第三阶段测试"
- 卡片B：徽章"1"，内容"制定项目计划" 
- 卡片C：徽章"高"，内容"重要任务提醒"
- 卡片D：徽章"2"，内容"开发核心功能"

选中这些卡片后使用功能，复制到剪贴板的内容将是：
```
[1] 制定项目计划

[2] 开发核心功能

[3] 完成第三阶段测试

[高] 重要任务提醒
```

## 排序规则

### 徽章类型优先级

系统自动识别徽章类型并按以下优先级排序：

1. **数字徽章**（优先级最高）
   - 示例：1, 2, 3, 10, 99
   - 排序：按数值大小升序排列

2. **文字徽章**（优先级中等）
   - 示例：高, 中, 低, A, B, C, TODO, DONE
   - 排序：按字母顺序排列

3. **Emoji徽章**（优先级最低）
   - 示例：🔥, ⭐, ✅, ❌, 🎯
   - 排序：按Unicode码点排列

### 混合类型排序示例

如果选中的卡片包含不同类型的徽章：
- 数字徽章：1, 2, 10
- 文字徽章：A, B, high
- Emoji徽章：🔥, ⭐

最终排序结果为：
```
1, 2, 10, A, B, high, 🔥, ⭐
```

## 技术特性

### 智能识别
- 自动识别徽章类型（数字/文字/Emoji）
- 只处理带有徽章的文本卡片
- 忽略空白内容或无徽章的卡片

### 异步处理
- 使用异步方法获取徽章信息
- 确保所有徽章数据获取完成后再排序
- 提供实时处理状态反馈

### 错误处理
- 检查Canvas环境可用性
- 验证选中卡片数量
- 提供详细的错误提示

## 适用场景

### 项目管理
- 按优先级（1,2,3）复制任务列表
- 按状态（TODO, DOING, DONE）整理工作流程
- 按重要性（🔥高, ⭐中, 💡低）排序需求

### 学习笔记
- 按章节号（1.1, 1.2, 2.1）复制学习内容
- 按重要性（A, B, C）整理知识点
- 按复习优先级（⭐⭐⭐, ⭐⭐, ⭐）安排学习计划

### 内容创作
- 按段落编号整理文章结构
- 按重要性排序写作要点
- 按发布顺序整理内容素材

## 注意事项

1. **徽章要求**：只有带徽章的文本卡片才会被包含在复制结果中
2. **选择范围**：需要选中多张卡片才能使用此功能
3. **内容过滤**：空白内容的卡片会被自动忽略
4. **格式保持**：原始文本内容的换行和格式会被保持

## 与其他功能的区别

| 功能 | 排序依据 | 输出格式 | 适用场景 |
|------|----------|----------|----------|
| 按位置复制 | 卡片在Canvas中的位置 | 纯文本内容 | 空间布局相关 |
| **按徽章顺序复制** | **卡片的徽章标记** | **[徽章] 内容** | **优先级/分类管理** |
| 普通复制 | 无排序 | 纯文本内容 | 简单内容获取 |

# 功能：查看和编辑卡片属性

## 用途

通过统一入口管理单张或多张文本卡片的尺寸、位置和其他基础属性。

相关代码：

- `src/presentation/commands/PropertiesCommands.ts`
- `src/presentation/modals/SingleCardPropertiesModal.ts`
- `src/presentation/modals/CardPropertiesModal.ts`
- `src/services/CardService.ts`

## 入口

- 单张文本卡片右键：`管理卡片属性`
- 多选文本卡片菜单：`管理卡片属性`
- 命令面板：`复制选中卡片的尺寸`

插件会根据当前选区数量自动打开对应的属性查看器。

## 单卡片模式

- 查看尺寸、位置、徽章和内容预览
- 单独调整宽度和高度
- 复制尺寸、位置或完整属性信息

## 多卡片模式

- 查看选区数量和尺寸范围
- 浏览每张卡片的基础属性
- 统一调整宽高
- 复制统计或尺寸信息

## 适用场景

- 检查某张卡片的实际尺寸和位置
- 批量统一卡片视觉规格
- 复制尺寸用于手动对齐其他卡片

# 功能更新说明

## 版本 1.2.0

### 新增功能

#### 1. 按标题拆分卡片
- **按标题层级拆分**：支持按 Markdown 的一级到六级标题拆分 Canvas 文本卡片。
- **多层级内容归属清晰**：选择某一级标题拆分时，更深层标题内容会保留在所属父级卡片中。
- **导言内容保留**：标题前的文本内容会自动并入第一张拆分后的卡片。

#### 2. 统一拆分入口
- **拆分卡片入口整合**：单卡片右键菜单中的拆分功能改为统一入口 `拆分卡片...`。
- **轻量选择器**：点击后弹出一个轻量选择器，集中展示当前卡片可用的拆分方式。
- **功能始终可见**：即使当前卡片不满足条件，`按分隔符拆分` 和 `按标题拆分` 也会保留在选择器中，并以禁用状态提示原因。

#### 3. 发布包优化
- **单文件下载**：发布时将 `main.js`、`manifest.json`、`styles.css` 打包为一个 ZIP 资源，避免用户分别下载三个文件。

### 使用说明

#### 单卡片拆分
1. 在 Canvas 中右键点击一个文本卡片
2. 选择 `拆分卡片...`
3. 在弹出的选择器中选择：
   - `按分隔符拆分`
   - `按一级标题拆分`
   - `按二级标题拆分`
   - 其他当前可用的标题层级

### 功能亮点

1. **菜单更清晰**：避免在一级右键菜单中持续堆积拆分功能
2. **反馈更直观**：不可执行的拆分方式仍然可见，减少“功能消失”的困惑
3. **扩展更容易**：后续新增其他拆分方式时，可直接纳入同一选择器
4. **发布更方便**：用户只需下载一个 ZIP 包即可安装插件

## 版本 1.1.0

### 新增功能

#### 1. 卡片属性管理功能
- **单卡片属性管理**：右键点击单个文本卡片，选择"编辑卡片尺寸..."可以查看和编辑卡片的尺寸、位置等属性。
- **多卡片属性管理**：选中多个文本卡片，右键选择"批量管理卡片属性..."可以统一管理多个卡片的尺寸和位置属性。

#### 2. 批量尺寸调整
- **统一尺寸选项**：支持将选中卡片统一为最小、最大或平均尺寸
- **自定义尺寸**：支持输入自定义尺寸值
- **单维度调整**：支持只调整宽度或只调整高度，保持另一维度不变
- **锁定宽高比**：支持锁定宽高比进行等比例调整

#### 3. 属性复制功能
- **尺寸复制**：可以复制单个或多个卡片的尺寸信息
- **位置复制**：可以复制卡片的位置信息
- **完整属性复制**：可以复制卡片的完整属性信息

### 界面改进

#### 1. 单卡片属性查看器
- 现代化界面设计，提供清晰的信息展示
- 集成尺寸编辑和复制功能
- 支持键盘快捷键操作

#### 2. 多卡片属性查看器
- 双栏布局设计，功能分区更清晰
- 统计信息展示，提供选中卡片的概览
- 表格形式展示卡片列表，支持悬停效果
- 批量操作按钮优化，提供更好的用户体验

### 技术改进

#### 1. 代码重构
- 分离单卡片和多卡片属性查看器，提供专门化的界面
- 优化CardService服务，添加单维度调整方法
- 改进错误处理和用户反馈机制

#### 2. 性能优化
- 批量操作优化，减少Canvas API调用次数
- 添加撤销机制，支持撤销最近的尺寸调整操作
- 优化数据处理流程，提高响应速度

### 使用说明

#### 单卡片操作
1. 在Canvas中右键点击单个文本卡片
2. 选择"编辑卡片尺寸..."
3. 在弹出的属性查看器中进行相关操作

#### 多卡片操作
1. 在Canvas中选择多个文本卡片
2. 右键点击选中的卡片
3. 选择"批量管理卡片属性..."
4. 在弹出的属性查看器中进行相关操作

### 功能亮点

1. **灵活的尺寸调整**：支持多种尺寸调整方式，满足不同使用场景
2. **直观的界面设计**：现代化界面设计，操作简单直观
3. **完整的属性管理**：集成了尺寸、位置、徽章等所有属性管理功能
4. **实用的复制功能**：支持多种复制选项，方便信息提取和分享
5. **可靠的撤销机制**：提供撤销功能，避免误操作带来的问题
